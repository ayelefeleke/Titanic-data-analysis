The analysis of this titatic data science is performed by using Python, Pandas, and Google Colab. The goal is to understand survival patterns based on gender, passenger class, age, and other features. The dataset contains information about Titanic passengers such as: Passenger class (Pclass) Gender (Sex) Age Survival status Analysis performed are Data loading and cleaning Handling missing values Exploratory Data Analysis (EDA) Survival comparison by: ->Gender ->Passenger class ->Age groups Visualization of survival trends The insights are Female passengers had higher survival rates than males. First-class passengers were more likely to survive. Children had better survival chances compared to adults
